# Self-Balancing-Robot
Self-balancing robots provide a practical platform for studying inverted pendulum control. This work presents a low-cost ESP32 &amp; MPU6050-based robot using PID control and filtering for tilt estimation. The robot achieves stable balance within 1.5–2 s and recovers from disturbances in 300–700 ms, demonstrating reliable real-time performance.
